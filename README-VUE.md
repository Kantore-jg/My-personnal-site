# Kantore Jean Gildas Portfolio

## Vue.js Version

A modern, responsive portfolio website built with Vue.js 3 and Vite.

### Features

- **Modern Vue.js 3** with Composition API
- **Responsive Design** that works on all devices
- **Component-Based Architecture** for maintainability
- **Smooth Animations** and transitions
- **Downloadable CV** functionality
- **Project Showcase** with image galleries
- **Social Media Integration**

### Quick Start

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm run dev
   ```

3. **Open in Browser**
   Navigate to `http://localhost:3000`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

### Project Structure

```
src/
├── components/
│   ├── Header.vue      # Fixed header with profile image
│   ├── Hero.vue        # Main hero section
│   ├── About.vue       # About section
│   ├── Skills.vue      # Skills showcase
│   ├── Projects.vue    # Featured projects
│   ├── Education.vue   # Education background
│   ├── Courses.vue     # Courses & certifications
│   ├── Contact.vue     # Contact information
│   └── Footer.vue      # Footer with social links
├── App.vue            # Main application component
└── main.js            # Application entry point
```

### Customization

- **Colors**: Modify CSS variables in component files
- **Content**: Update data properties in Vue components
- **Images**: Replace images in the `images/` folder
- **Contact Info**: Update links in Footer.vue

### Technologies Used

- Vue.js 3
- Vite (Build Tool)
- CSS3 with modern features
- Responsive Grid Layout
- Google Fonts
