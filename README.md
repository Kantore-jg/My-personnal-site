# Kantore Jean Gildas - Portfolio Professionnel

Portfolio moderne et professionnel développé avec Vue.js 3, présentant mes compétences, projets et expériences en développement FullStack.

## 🚀 Fonctionnalités

- **Design Moderne** : Interface professionnelle et responsive
- **Vue.js 3** : Application moderne avec composants réutilisables
- **Navigation Smooth** : Navigation fluide entre les sections
- **Téléchargement CV** : CV disponible au format PDF
- **Formulaire de Contact** : Formulaire fonctionnel avec validation
- **Galerie de Projets** : Images du projet E~Menage
- **Intégration Sociale** : Liens vers tous les réseaux sociaux
- **Responsive Design** : Optimisé pour tous les appareils

## 🛠️ Technologies

- **Frontend** : Vue.js 3, Vite
- **Styling** : CSS3 avec variables CSS
- **Build** : Vite build system
- **Deployment** : GitHub Pages compatible

## 📦 Installation

1. Clonez le dépôt :
```bash
git clone git@github.com:Kantore-jg/My-personnal-site.git
cd My-personnal-site
```

2. Installez les dépendances :
```bash
npm install
```

3. Démarrez le serveur de développement :
```bash
npm run dev
```

4. Ouvrez `http://localhost:3000` dans votre navigateur

## 🌐 Déploiement

### GitHub Pages

1. Installez les dépendances :
```bash
npm install
```

2. Build pour la production :
```bash
npm run build
```

3. Déployez sur GitHub Pages :
```bash
npm run deploy
```

Ou configurez GitHub Pages pour servir depuis la branche `main` avec le dossier `dist`.
